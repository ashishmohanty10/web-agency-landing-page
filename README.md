## Installation

```bash

git clone git@github.com:ashishmohanty10/web-agency-landing-page.git

# Enter folder
cd your-repo-name

# Install dependencies
npm install or pnpm i
```

## Running the Project

```bash

npm run dev or pnpm dev

# Your site will be available at:

http://localhost:3000
```

## Production Build

